# Exemplo de uso da biblioteca PlUpload em ASP.Net MVC 4 #

	Dois exemplo de como utilizar o PlUpload em ASP.Net MVC 4

## Primeiro Exemplo ##
No exemplo personalizado é demonstrado com criar dois links (Selecionar Arquivo e Enviar Arquivos)
Exibi um progresso do envio dos arquivos.

## Segundo Exemplo ##
No exemplo completo é mostrada a própria interface do PLUpload, a versão completa que exibi uma grid, onde você pode escolher os arquivos a serem enviados, remover os arquivos da lista, e por fim enviar os arquivos
